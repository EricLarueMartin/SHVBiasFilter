Detector bias filtering

SHV Bias Filter

A compact, axisymmetric high-voltage low-current filter for LEGEND-style detector biasing and related experiments.

Simulation Focus

  • Rounded inner edge of each ground plate.
  • Rounded outer edge of each high-voltage plate.
  • Field effects from the optional outer grounded tube.
  • Dielectric boundaries between the conductive core, washer material, and epoxy fill.

Simulation Results

  • Report peak electric field in the dielectric, not inside conductor cells.
  • Show where the peak occurs in the r-z half-section.
  • Compare washer and epoxy peak fields to material breakdown placeholders.
  • State whether the slide is showing the latest solved result or a preview solve.
  • Use this as a screening result until the conforming FEA backend is validated.

Analytic Sanity Checks

  • Use V/d for each local clearance as the baseline stress.
  • Use r as the minimum exposed conductor radius and d as the minimum gap.
  • Bound the screening field by Emax = V(r + d)/(r d).
  • Keep local 2D and coax checks as lower-order comparisons.
  • Show Rmaj/r so toroidal-curvature limits are visible.
  • Compare both radial sides: HV disc to tube and ground disc to HV core.
  • Use FEA to confirm the actual ring geometry respects the radius and gap assumptions.

Radius-Gap Limit

  • Let r be the minimum radius on any exposed conductor feature.
  • Let d be the minimum dielectric gap to the opposite conductor.
  • The limiting geometry is a radius-r sphere inside a radius-(r + d) grounded sphere.
  • Use Emax = V(r + d)/(r d) as the screening field bound.
  • Use FEA to check the actual geometry and catch any smaller-radius features.

Field-Limited Packing

  • Assume plate thickness is 2r and dielectric gap is d.
  • Use the radius-gap limit for the most limiting geometry: Emax = V(r + d)/(r d).
  • Use gap capacitance proportional to 1/d.
  • Use stack pitch L = d + 2r.
  • Figure of merit: FOM = C/(L Emax).
  • The shape optimum is r/d = 1/√2.

RC Ladder

  • Estimate HV-ground capacitance from the washer overlap area.
  • Estimate bias-to-bias parasitic capacitance through the selected core material and epoxy around the core.
  • Track the high-frequency capacitive feedthrough ratio Cpar/Cg.
  • Mark where parasitic reactance equals the bias-to-bias resistance.
  • Plot voltage-noise attenuation in dB using 20 log10.

FEA Capacitance Sanity Check

  • Compare analytic washer-overlap Cg,total to FEA energy-extracted Cg,total.
  • Compare analytic adjacent-bias Cpar to the FEA three-bias energy-difference estimate.
  • Keep attenuation validation on the loaded RC and SPICE ladder slides.
  • MELF Cpar uses an effective ceramic core: εeff = εsubstrate/(1 - FF).

Loaded RC Ladder

  • Solve the RC ladder as individual R, Cg, and Cpar elements.
  • Place the expected load only at the output node.
  • Use Gload = Iload/Vbias.
  • Use Cload = Cdetector + length/(Z0 vf c) for lumped coax capacitance.

SPICE Ladder Backend

  • Backend solves a SPICE-style small-signal AC netlist.
  • RC ladder uses individual R, Cg, and Cpar elements.
  • Coax is treated as a lossless transmission line with Z0, delay, detector C, and load conductance.
  • Compare full transmission-line solve against lumped coax capacitance and resistor-section one-stage estimates.

Prototype Materials

  • Alumina washer dielectric is the preferred prototype target.
  • FR4/G10 remains the easiest fallback if suitable alumina washers are unavailable.
  • MG Chemicals 9510 heat-cure epoxy as the first potting-fill option.
  • Selectable core/conductor material presets, with Fair-Rite type 61 NiZn ferrite as the first sample set.
  • Copper is the default plate and ground-tube material for RF-loss and skin-depth estimates.
  • Conductive epoxy joints between core discs and bias plates.

Construction Plan

  • Stack ferrite slices, polished copper bias discs, ground discs, ceramic washers, and mechanical spacers with epoxy flow holes.
  • Use conductive epoxy at ferrite-to-bias-disc contacts before final potting fixes the stack.
  • Fill with heat-cured epoxy, then use repeated vacuum and pressure dwell cycles to reduce trapped voids.
  • Degas controller subproject: two relay-driven valves switch a pressure vessel between vacuum pump and pressure source.
  • Controller inputs: high pressure, low pressure, pressure bands, dwell time at each band, and repeat count.
  • Pressure-cure integration in an oven is possible later, but outside the first prototype scope.
